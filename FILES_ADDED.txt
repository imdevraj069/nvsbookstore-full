🔧 CONFIGURATION
═══════════════════════════════════════════════════════════════════════════

Cache TTL (Time To Live):
  - Products & Notifications:  5 minutes (default)
  - Single items:             10 minutes (default)
  - Edit in: src/lib/api-service.js

Disable Cache for Testing:
  - Use skipCache = true parameter in any fetch function
  - Example: apiService.getProducts({}, true)

Clear All Cache:
  - In browser console: localStorage.clear()
  - In code: cache.clear()

