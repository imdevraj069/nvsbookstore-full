4. CLOUDFLARE TUNNEL CONFIGURATION
═══════════════════════════════════════════════════════════════════════════════

INSTALLATION:
  macOS:   brew install cloudflare/cloudflare/cloudflared
  Linux:   curl -L ... | tar -xvf ... && sudo mv cloudflared /usr/local/bin/
  Windows: Download from GitHub releases

SETUP STEPS:
  1. cloudflared tunnel login
     → Opens browser to authenticate with Cloudflare

  2. cloudflared tunnel create nvs-bookstore
     → Creates tunnel, saves credentials to ~/.cloudflared/UUID.json

  3. Update ~/.cloudflared/config.yml
     → Copy template from: cloudflare-tunnel-config.yml
     → Replace "your-uuid" with actual UUID
     → Replace "yourdomain.com" with your domain

  4. Add DNS Records in Cloudflare Dashboard:
     app.yourdomain.com            CNAME  [UUID].cfargotunnel.com
     api.yourdomain.com            CNAME  [UUID].cfargotunnel.com
     transaction.yourdomain.com    CNAME  [UUID].cfargotunnel.com
     read.yourdomain.com           CNAME  [UUID].cfargotunnel.com
     admin.yourdomain.com          CNAME  [UUID].cfargotunnel.com

  5. Start Tunnel:
     cloudflared tunnel run nvs-bookstore
     (or: cloudflared service install && cloudflared service start)

  6. Update Environment for Production:
     apps/web/.env.local:
       NEXT_PUBLIC_API_URL=https://api.yourdomain.com
     
     services/transaction-service/.env:
       MONGO_URI=mongodb+srv://nvsbookpdf:hxzD9Sc2rKXjMq4u@cluster0.hmclrwp.mongodb.net/nvsbookstore

  7. Update Google OAuth Authorized Domains:
     Google Cloud Console → APIs & Services → Credentials → Edit OAuth Client
     
     Authorized Origins:
       - https://app.yourdomain.com
       - https://yourdomain.com
     
     Authorized Redirect URIs:
       - https://app.yourdomain.com/auth/login
       - https://app.yourdomain.com/auth/signup
